# 🌸 Pink Galaxy — Arch Linux Installation Guide

Interactive step-by-step Arch Linux installation guide for Dell Inspiron 14 7420 2-in-1.

## Files
- `index.html` — Main guide (all 16 chapters)
- `css/style.css` — Pink Galaxy theme
- `js/app.js` — Interactive features
- `amplify.yml` — AWS Amplify deployment config

## Features
- ✅ Progress checkboxes (saved to localStorage)
- 📖 Learning Notebook (floating panel)
- 🖊 Draggable sticky Post-Its
- 🔀 Alternative paths (btrfs/ext4, wipe methods, keyboard layouts)
- 🎯 Focus Mode
- ⌨️ Keyboard navigation (← → chapters, N notebook, F focus)
- 📋 Copy buttons on all commands
- 💾 Auto-save progress

## Deploy to AWS Amplify
1. Push this folder to a GitHub repo
2. Connect repo to AWS Amplify
3. Amplify auto-detects static site — no build needed
4. Done!
